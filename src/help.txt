Usage: neocities-deploy [--source=<dir>] [--dest=<url>] [--ignore=<file>]... [-h|--help] [--version]

Options:
    --source <dir>      Source directory. Default to './'.
    --dest <url>        Destination URL. Defaults to root.
    --ignore <file>     Exclude file(s) in the Source directory.
    -h --help           Display help.
    --version           Display version.
